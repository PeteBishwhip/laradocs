# Content

Laradocs ships with content features that go beyond plain Markdown:

- [Rich Content](/docs/content/rich-content) — callouts, code blocks, media embeds and more.
- [Variables](/docs/content/variables) — interpolate dynamic values into your docs.
- [Macros](/docs/content/macros) — reusable content snippets.
- [Components](/docs/content/components) — Blade-style `<x-...>` tags in Markdown.
- [Icons](/docs/content/icons) — inline icons and navigation icons using Heroicons or a custom set.
