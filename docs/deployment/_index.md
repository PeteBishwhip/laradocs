# Deployment

Laradocs is a standard Laravel package — deploying it means deploying your Laravel application. Pick the guide that matches your setup:

- [Laravel Forge](/docs/deployment/forge) — provision a VPS and deploy with Forge's dashboard and automated deploy scripts.
- [Laravel Cloud](/docs/deployment/cloud) — fully managed platform with autoscaling, zero-downtime deploys, and managed queues.
- [Custom VPS](/docs/deployment/vps) — self-hosted checklist for any server you manage yourself.
